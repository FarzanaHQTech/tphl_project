Introduction

Thank you for purchasing the Manez - HRM & CRM HTML5 Dashboard Template.

Manez is a simple and beautiful admin dashboard template built on top of awesome Bootstrap 5.3.3. It's crafted using modern web technologies HTML5, CSS3, and jQuery.

Manez is a beautifully crafted, clean, and minimal designed admin template with Dark, Light Layouts and RTL options. It is specifically designed for HRM (Human Resource Management) and CRM (Customer Relationship Management) applications. You can build any type of HRM and CRM web application, like employee management systems, customer interaction platforms, and more.

Manez Admin & Dashboard Template is easily customizable and has developer-friendly codes. It will help your team move faster and save development costs and valuable time.

If you’re a developer looking for an HRM & CRM admin dashboard that is fully responsive with Bootstrap, then you are at the right place to start your project using Manez Admin & Dashboard Template.

Manez contains lots of new design widgets with responsiveness on all screens. It is very easy to change any layout in your existing running application. We have written minimum SCSS and codes to increase performance.
